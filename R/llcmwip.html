Der Datensatz

                         delim = ";", 
                         escape_double = FALSE,
                         show_col_types = FALSE,
                         col_names = TRUE,

Vertikale Position

Wir erzeugen eine grafische Darstellung der vertikalen Positionen, um die Bewegungen zu visualisieren.

posplot <- ggplot(llcmwip) + 
  geom_line(mapping=aes(x = Zeit, 
                         y = Position)) +
  labs(
         title="Vertikale Position eines Controllers im Simulator",
         x = "Zeit in Sekunden",
         y = "Vertikale Position in Metern"
  )

Moving Average für die Geschwindigkeiten

Wir verwnden die Funktion rollmean im Package zoo in einer Pipeline. die Variable k legt fest, wie viele Werte wir verwenden für den Moving Average.

k <-815
llcmwip <- llcmwip %>%
  mutate(smoothedvelocity = rollmean(Signalgeschwindigkeit, k=k, fill = NA))

ggplot(llcmwip) + 
  geom_line(mapping=aes(x = Zeit, 
                         y = smoothedvelocity)) +
  labs(
         title="Signalgeschwindigkeit mit finiten Differenzen",
         x = "Zeit in Sekunden",
         y = "Signalgeschwindigkeit (Moving Average)"
  ) 
## Warning: Removed 814 rows containing missing values (`geom_line()`).

# Loess-Smoother aus ggplot2

span <- 0.3

loessplot <- ggplot(llcmwip, aes(Zeit, Signalgeschwindigkeit)) + 
  geom_smooth(method='loess', 
              se = FALSE,
              span=span) +
  labs(
         title="Geglättete Signalgeschwindigkeit mit finiten Differenzen",
         x = "Zeit in Sekunden",
         y = "Signalgeschwindigkeit (Loess Smoother)"
  )

Drei Plots als eine Abbildung

Wir erzeugen am Ende noch eine Darstellung, bei der wir alle drei Plots, die wir ins Buch bringen möchten als eine Abbildung arrangieren. Dazu verwenden wir die Funktion ggarrange aus dem Package ggpubr.

ggarrange(posplot, velplot, loessplot,
          common.legend = TRUE,
          heights=c(4,5,5),
          ncol = 1, nrow = 3)

ggsave(filename="wipall.png",
       plot=last_plot(), 
       device="png",
       path="images/", 
       width=16, 
       height=9, 
       units="cm")
