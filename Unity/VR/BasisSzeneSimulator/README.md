# Basis-Szene im VIU-Simulator

Dieses Unity-Projekt enthält die Basis-Szene, wie schon aus den Desktop-Versionen bekannt. Das Projekt enthält das VIU-Plugin, in den Preferences
ist der Simulator aktiviert.
